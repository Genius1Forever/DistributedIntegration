# Distributed Integration System (C++)

Клиент-серверная система для распределённого численного интегрирования функции:

1 / ln(x)

Проект реализован на C++ (стандарт C++17) с использованием многопоточности и сетевого взаимодействия.

---

## Возможности

- Клиент-серверная архитектура
- Распределение вычислений между клиентами
- Использование всех ядер CPU
- Многопоточные вычисления
- Логирование работы
- Кроссплатформенность (Windows / Linux)
- Сборка через CMake

---

## Метод вычислений

Для численного интегрирования используется метод трапеций.

Формула:

∫ f(x) dx ≈ Σ (f(x) + f(x+h)) / 2 · h

Где:

f(x) = 1 / ln(x)

Дополнительно реализован метод Симпсона.
Метод Симпсона обеспечивает более высокую точность по сравнению с методом трапеций.

## Запуск программы

1. Запуск сервера

В отдельном окне терминала:

server.exe
или
./server

После запуска введите:

1) метод интегрирования: 
	1 - Метод трапеций
	2 - Метод Симпсона
	> 1 или 2;
2) нижний предел a;
3) верхний предел b;
4) шаг h.

Пример:

Введите a: 2
Введите b: 5
Введите h: 0.001

2. Запуск клиентов

В других окнах терминала запустите:

client.exe
или
./client

Количество клиентов можно запускать несколько раз (по умолчанию максимум 2, можно изменить в коде).

---

## Принцип работы

1. Сервер принимает подключения клиентов
2. Клиенты передают количество ядер CPU
3. Сервер делит диапазон интегрирования
4. Сервер отправляет задания клиентам
5. Клиенты выполняют вычисления в потоках
6. Клиенты возвращают результат
7. Сервер суммирует данные и выводит итог
